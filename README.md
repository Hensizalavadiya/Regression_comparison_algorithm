# Regression_comparison_algorithm

## Overview
This project focuses on applying multiple machine learning regression algorithms to a dataset and comparing their performance.

## Objective
The goal of this project is to evaluate different regression models and identify which algorithm performs best for predicting continuous values.

## Algorithms Used
- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor
- KNeighborsRegressor
- GradientBoostingRegressor
- Support Vector Regressor

## Tools and Libraries
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Conclusion
Different regression algorithms were applied and evaluated using performance metrics to determine the most effective model for the dataset.
